Project 1

Describe how search engines work

Search engines are automated and use crawling and indexing to browse the web to find the information that is most relevant to your search. Search engines do this through links in order to reach the billions of documents on the web.
Once the engines find these pages, they decipher the code from them and store selected pieces in massive databases, to be recalled later when needed for a search query. To accomplish the monumental task of holding billions of pages that can be accessed in a fraction of a second, the search engine companies have constructed datacenters all over the world. These storage facilities hold thousands of machines processing large quantities of information quickly.

Explain the importance of keywords and descriptions as meta tags

Keywords help search engines find relevant pages and information to your search. The search engine keeps track of keyword-based indexes as they crawl and index the web.
Meta tags include meta robots and meta description. Meta robots can be used to control search engine crawler activity per-page.
Meta description tag are short descriptions of the page's content that shows up in the search engine. Search engines do not use the keywords or phrases in this tag for rankings.

Four elements that comprise writing relevant content links

  1. Links should have good anchor text. Anchor text tells the user and search engines something about the page the link goes to before visiting the link.
  2. Links should always link to trusted sources.
  3. Links should bring traffic to your site.
  4. Links should be relevant. They should be contected to the page content and relevant to the user.

What constitutes good heading hierarchy?

Good heading hierarchy uses consecutive header tags (h1-h6) to create a nested outline of content for a webpage. In practice, headings should never skip levels. For example, a h2 should not be followed by an h5. There should only be a single h1 heading per page, nesting should be logical (i.e., h2 with three h3 and an h4). Headings should be meaningful and consice.
